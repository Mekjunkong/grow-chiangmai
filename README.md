# Grow Chiang Mai

A static landing page and market note for a Chiang Mai garden-membership concept with live plot cameras, managed and DIY growing plots, an AC farm cafe, local compost sourcing, and tool rental.

## Website

- Live Vercel deployment: https://goal-facebook-com-reel-129400390955.vercel.app
- GitHub repository: https://github.com/Mekjunkong/grow-chiangmai
- Static site file: `outputs/grow-chiangmai/index.html`
- Market research note: `outputs/grow-chiangmai/market-research.md`

## Local build

```bash
npm install
npm run build
```

The Vercel project is configured with:

- Build command: `npm run build`
- Output directory: `outputs/grow-chiangmai`

## Product Direction

Grow Chiang Mai is structured as a Thai-operated farm service, not a foreign land ownership product. Members subscribe to garden access, care, learning, live bed cameras, local compost, and harvest options.
